# Catbas (Catálogos Generales)
El paquete Catbas (Catálogos Generales) de V12 es un grupo de aplicaciones que sirven como base para administrar los catálogos compartidos por distintos módulos del ERP V12. Su función principal es centralizar y estandarizar la información que se utiliza en procesos de compras, ventas, inventarios, contabilidad y tesorería, garantizando coherencia y consistencia en toda la operación.
